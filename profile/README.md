# Pixly Stream

> Pixel art collaboratif live pour streamers Twitch.

Pixly Stream permet à une communauté de streamer de poser des pixels en temps réel sur la grille de son streamer favori, comme r/place mais centré sur le live et la commu Twitch.

**Statut** : Pre-MVP — en construction.

## Stack

- **Backend** — Go (centrifuge + Redis + Postgres + pgx)
- **Frontend** — React + Vite + TypeScript + Tailwind + Canvas API
